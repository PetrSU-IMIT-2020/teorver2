# Билет №34. Асимптотически эффективная стат. оценка.

$\theta_n^{\*}$ - асссимптотически эффективная, если

$\lim\limits_{n \to \infty}{\frac{M[(\theta - \theta_n^{\*})^2]}{\underset{\theta_n^{\*\*}}{inf} \ M[(\theta - \theta_n^{\*\*})^2]}} = 1$

$inf$ - точная нижняя граница
